# PyPad - secure notepad

PyPad is a university project - the tiny copy of default Windows notepad build with Python and Tkinter.

#### Functionality:

 - **Program:**

   + Opening files
   + Saving files
   + Closing editor
   + Choosing directory

 - **Editor:**
  
   + Repalce
   + Find
   + Line numbering
   + Word count

 - Security:
   
   + Encrypt file
   + Decrypt file
